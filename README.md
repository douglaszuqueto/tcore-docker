# Docker image for TagoCore platform

Docker image for TagoCore platform

## Support

* X64 Systems

## Versions

* Latest
* v0.4.1

## Install

```bash
docker run -d --name tcore -p 8080:8888 douglaszuqueto/tcore:v0.4.1
```

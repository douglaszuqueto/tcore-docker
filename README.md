# Docker image for TagoCore platform

Docker image for TagoCore platform

## Support

* X64 Systems

## Versions

* v0.4.1
* v0.5.0
* Latest

## Install

```bash
docker run -d --name tcore -p 8080:8888 douglaszuqueto/tcore:v0.5.0
```
